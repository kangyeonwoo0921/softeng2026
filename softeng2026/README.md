## GitHub Pages 설정
1. 이 리포지토리를 GitHub에 푸시하세요.
2. 리포지토리 설정(Settings) > Pages로 이동.
3. Source를 "Deploy from a branch"로 설정.
4. Branch를 main, 폴더를 /docs로 선택.
5. Save를 클릭하면 GitHub Pages가 활성화됩니다.
6. 몇 분 후에 https://[username].github.io/[repository-name]/ 에서 접속 가능합니다.
